# first steps

## commands

| command                   | action                                    |
| :------------------------ | :---------------------------------------- |
| jj git init               | initialize the repository                 |
| jj st                     | show the repository status                |
| jj desc                   | create (update) the change description    |
| jj git remote add origin  | add a remote server                       |
| jj git push               | push to the remote server                 |
